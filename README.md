An AI‑based project that uses deep learning and computer vision techniques to classify and identify medicinal plants from images. This project aims to support botanical research, aid in the preservation of medicinal species, and enable easy recognition through automated classification.

Key Features:

🖼️ Trained a deep learning model to recognize various medicinal plants from images

🌱 Enables accurate classification and labeling of plants for research and conservation

⚡ Provides a foundation for mobile or web applications for on‑the‑go plant identification

🐍 Built with Python, TensorFlow / Keras, and OpenCV
